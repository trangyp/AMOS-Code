# Changelog

All notable changes to AMOS will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [2.3.0] - 2024-04-15

### Added

- **Configuration Management**: Pydantic Settings v2 with environment support
- **Observability**: OpenTelemetry tracing and Prometheus metrics
- **API Documentation**: OpenAPI 3.0 with Swagger UI and ReDoc
- **CI/CD**: 7 GitHub Actions workflows for testing, linting, security
- **Documentation Site**: MkDocs Material documentation

### Enhanced

- **API Server**: Enhanced FastAPI with comprehensive documentation
- **Security**: JWT authentication, RBAC, rate limiting
- **Testing**: 36+ automated tests with pytest

## [2.2.0] - 2024-03-20

### Added

- **Vector Memory**: ChromaDB integration with RAG capabilities
- **Authentication System**: JWT-based auth with role-based access control
- **Testing Framework**: Comprehensive pytest-based test suite
- **Security Scanning**: Bandit, Safety, TruffleHog, Trivy integration

### Enhanced

- **Memory System**: 5-tier memory with episodic and semantic storage
- **Dashboard**: Streamlit 7-page web dashboard

## [2.1.0] - 2024-02-15

### Added

- **MCP Tools**: 10 real-world MCP tools (filesystem, git, web search, etc.)
- **Docker Compose**: 14-service microservices setup
- **Self-Evolution**: 5-phase autonomous code improvement
- **Repo Doctor Omega**: 27-dimension repository pathology monitoring

### Enhanced

- **Orchestrator**: Hybrid neural-symbolic reasoning engine
- **Unified System**: 8-subsystem orchestration hub

## [2.0.0] - 2024-01-10

### Added

- **Hybrid Cognitive Engine**: Neural + Symbolic + Hybrid paradigms
- **Global Laws L1-L6**: Safety constraint system
- **Multi-Agent System**: Architect, reviewer, auditor, executor agents
- **Production API**: FastAPI REST and WebSocket interfaces
- **16 LLM Providers**: Local (Ollama) and cloud provider support
