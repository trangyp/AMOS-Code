# User Guide

Learn how to use AMOS effectively for building intelligent agent systems.

## What's in This Guide

This guide covers everything you need to know to use AMOS productively:

- **Agents**: Understanding and creating specialized agents
- **Orchestration**: Coordinating multiple agents for complex tasks
- **Memory**: Using AMOS's tiered memory system effectively
- **Laws**: Understanding Global Laws L1-L6
- **Tools**: Using MCP tools to interact with the world
- **Security**: Authentication, authorization, and best practices

## Prerequisites

Before diving in, make sure you have:

1. AMOS installed ([Installation Guide](../getting-started/installation.md))
2. Basic understanding of Python
3. Familiarity with REST APIs (for API usage)

## Getting Help

If you get stuck:

- Check the [API Reference](../api-reference/index.md) for detailed function documentation
- Browse [GitHub Issues](https://github.com/trangyp/AMOS-Code/issues) for common problems
- Join [GitHub Discussions](https://github.com/trangyp/AMOS-Code/discussions) for questions
