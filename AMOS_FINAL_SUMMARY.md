## 🚀 Quick Start Guide

### **Option 1: Interactive Shell (Fastest)**
```bash
python amos_shell.py

# Inside shell:
> /ask What is the best architecture for microservices?
> /query scalability patterns
> /countries
> /status
> /exit
```

### **Option 2: Web Dashboard (Visual)**
```bash
# Terminal 1: Start API
python amos_api_enhanced.py

# Terminal 2: Open dashboard
open amos_dashboard.html

# Access at http://localhost:5000
```

### **Option 3: Docker Deployment (Production)**
```bash
docker-compose up -d

# Access API at http://localhost:5000
# Access dashboard at http://localhost:5000/dashboard
```

### **Option 4: API Direct Access**
```bash
# Query knowledge
curl -X POST http://localhost:5000/query \
  -H "Content-Type: application/json" \
  -d '{"query": "scalability"}'

# Ask brain
curl -X POST http://localhost:5000/think \
  -H "Content-Type: application/json" \
  -d '{"problem": "Best architecture?"}'
```
