# AMOS Brain - 统一认知架构 AI 代理

**一个具有结构化推理、认知合规和多代理编排的综合性 AI 代理框架。**

---

## 🧠 什么是 AMOS Brain？

AMOS Brain 是一个统一认知架构，为 AI 代理添加结构化推理能力：

- **Rule of 2** - 双视角分析（L2 推理）
- **Rule of 4** - 四象限系统思维（L3 推理）
- **6 全球法则** - L1-L6 合规执行
- **12 领域引擎** - 路由到专门智能
- **记忆与召回** - 持久化推理历史
- **分析仪表板** - 决策模式洞察
- **多代理编排** - 协调代理团队

---

## 🚀 快速开始

```bash
# 统一启动器 - 发现所有功能
python amos_brain_launcher.py

# 或从教程开始
python amos_brain_tutorial.py

# 或使用 CLI
python amos_brain_cli.py
```

查看 `amos_brain/README.md` 和 `AMOS_BRAIN_GUIDE.md` 获取完整文档。

## 📦 项目结构

| 组件 | 描述 |
|------|------|
| `amos_brain/` | 具有认知架构的核心大脑实现 |
| `clawspring/` | 支持多提供商的 AI 助手运行时 |
| `AMOS_ORGANISM_OS/` | 生物体级操作系统 |
| `memory/` | 持久化内存系统 |
| `skill/` | 具有内置功能的技能系统 |
| `multi_agent/` | 多代理编排 |
