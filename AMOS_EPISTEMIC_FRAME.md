## 2. The Seven Axioms of AMOS Cognition

### **Axiom 1 — No Direct Access to Reality-in-Itself**

The system never accesses reality-in-itself directly. It only receives bounded input, filtered signal, structured sensory or inferential data.

For any externally originating state $X$, the system only accesses:

$$
\boxed{
X_{\Sigma} = \Phi_{\Sigma}(X)
}
$$

Where $\Phi_{\Sigma}$ is the system-dependent transformation function.

Thus:

$$
\boxed{
Access(X) \neq X
}
$$

**First Law:** No direct access. Only mediated access.

---

### **Axiom 2 — The System Must Model**

Given any non-null input, the system must generate distinctions, groupings, and relations in order to operate.

$$
\boxed{
\mathcal{I}_t \neq \varnothing \Rightarrow \exists M_t \in \mathcal{M}
}
$$

Without model construction, there is no stable perception, no experience, and no action selection.

Thus cognition is not passive reflection. It is necessary model-generation:

$$
\boxed{
Cognition = ModelConstruction(\mathcal{I}_t, \mathcal{S})
}
$$

Therefore:

$$
\boxed{
Perception \neq ReflectionOfReality
}
$$

Instead:

$$
\boxed{
Perception = OperationalModeling
}
$$

### **Axiom 3 — All Measurement is Internal to the System**

The system can only measure using its own instruments:
- its sensors
- its concepts
- its logic
- its language
- its internally constructed tools

Thus every measurement is system-internal:

$$
\boxed{
Measure_{\Sigma}(x) = f_{\Sigma}(x)

$$

There is no measurement outside the system-relative framework:

$$
\boxed{
\not\exists Measure_{outside;\Sigma}(x);available;to;\Sigma

Hence all measurement is structurally self-referential:

$$
\boxed{
Measurement_{\Sigma} \subseteq SelfBoundedOperations_{\Sigma}
}
$$

Not every possible model can be generated by the system. The set of available models is constrained by:

So the reality the system can know is jointly determined by:

$$
\boxed{

$$
